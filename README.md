# gpt_app

```sh
$ cp sample.env .env  ## put the ChatGPT secret key in the .env
$ docker build -t gpt_app .
$ docker run -it --rm gpt_app "カレーライスを作りたいです。一般的なカレーライスの材料を教えてください。"
入力メッセージ： カレーライスを作りたいです。一般的なカレーライスの材料を教えてください。
カレーライスを作るために一般的に使用される材料は以下の通りです。

1. カレールウ（市販のものでも、自家製でも構いません）
2. ご飯（白米、玄米、もち米など）
3. 玉ねぎ
4. 人参
5. じゃがいも
6. 肉（鶏肉、豚肉、牛肉など。お好みで選びます）
7. 油（サラダ油、オリーブ油、バターなど）
8. 水または出汁（市販のものや、昆布、かつお節から取った出汁でも構いません）

追加でカレーに入れたい具材は、以下のものがあります：
1. ピーマン
2. ズッキーニ
3. エリンギ
4. シメジ
5. パプリカ
6. カボチャ
7. 枝豆

これらの材料を使って、美味しいカレーライスを作ることができます。具材はお好みでアレンジして楽しんでください。

```
